# Weonne Template Base

Este repositório contém o template base para iniciar um projeto VTEX-IO do zero, simplificando o setup inicial e agilizando o desenvolvimento de soluções na plataforma VTEX.

## Descrição

O **Weonne Template Base** oferece uma estrutura inicial com suporte para várias features da plataforma VTEX, como backend, checkout, componentes customizados e a store. Ele serve como ponto de partida para a construção de lojas online robustas, escaláveis e altamente personalizáveis.

### Features

O template base inclui as seguintes funcionalidades principais:

- **Backend:** Configurações e integração do backend para suporte de dados e lógica de negócios personalizada.
- **Checkout:** Customizações do fluxo de checkout da VTEX, permitindo uma experiência de compra personalizada para o usuário final.
- **Components:** Conjunto de componentes reutilizáveis prontos para serem utilizados e customizados conforme as necessidades do projeto.
- **Store:** Estrutura inicial da loja, pronta para receber ajustes de layout, design e funcionalidades específicas da marca ou cliente.

### Estrutura de Diretórios

O projeto está organizado da seguinte forma:

```bash
├── backend/               # Código e configurações do backend (Node.js, GraphQL, etc.)
├── checkout/              # Customizações do checkout da VTEX
├── components/            # Componentes reutilizáveis da loja
├── store/                 # Estrutura e configurações da loja (templates, styles, etc.)
├── README.md              # Documentação inicial do projeto
└── manifest.json          # Configurações principais do projeto VTEX-IO
